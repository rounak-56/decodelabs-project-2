# **DecodeLabs Summer Internship - Project 2**

A sleek, immersive landing page built around the highly anticipated release of Grand Theft Auto VI. This project serves as the second milestone for the DecodeLabs frontend internship, transitioning from fixed-dimension layouts (Project 1) to a fully responsive, Mobile-First architecture.

> **[View Live Demo Here](https://rounak-56.github.io/Decodelabs-project-2/)**

<br>

## Project Objective
The main goal of this project was to build a website that naturally adapts to any screen size. Whether you're viewing it on a massive desktop monitor or a phone, the layout is designed to adjust perfectly. I built it entirely from scratch using pure CSS to really nail down the fundamentals, rather than relying on shortcuts like Bootstrap or Tailwind.

<br>

## Tech Stack & Skills Highlighted
* **Mobile-First CSS:** Base styles engineered for mobile devices, using `@media` queries to scale up complexity for larger viewports.
* **Fluid Typography:** Utilized the CSS `clamp()` function to ensure text and button padding scale organically like a rubber band, preventing layout breaks.
* **Modern DOM APIs:** Implemented a JavaScript-free hamburger dropdown menu utilizing the native HTML `popover` attribute to keep the codebase lightweight and performant.
* **Immersive UI/UX:** Applied `100vh` dynamic viewport sizing, fixed background attachments, and sleek, semi-transparent neon overlays to capture a modern gaming aesthetic.
